# Mini-Slot Game (PixiJS + TypeScript) 🎰

Це тестове завдання з реалізації міні-слота з використанням PixiJS та TypeScript. Проект побудований за суворою архітектурою **MVC (Model-View-Controller)** для чіткого розділення ігрової логіки та відображення, що забезпечує легку підтримку та масштабованість.

### 🚀 Реалізований функціонал

- **Сітка 5x3:** Генерується динамічно на основі конфігурації.
- **Data-Driven логіка:** Результати спінів, лінії виплат та початкові налаштування (баланс, ставка) завантажуються безпосередньо з файлу `spins.json`.
- **Динамічна підсвітка:** Візуальне виділення виграшних ліній.
- **Інтерфейс (UI) у реальному часі:** Відображення суми виграшу та реактивне оновлення балансу.
- **Інтерактивність:** Управління станом кнопки Spin (блокування під час обертання).

### 🛠 Технологічний стек

- **Vite** — Швидке збирання проекту та середовище розробки.
- **PixiJS (v8)** — Високопродуктивний WebGL рушій для 2D графіки.
- **TypeScript** — Сувора типізація ігрової логіки, компонентів інтерфейсу та JSON-структур.

### 🏗 Архітектура (MVC)

- **Model (`src/core/Model.ts`):** Керує ключовим станом гри (баланс, ставка, поточна матриця, дані останнього виграшу). Повністю ізольована від PixiJS.
- **View (`src/core/View.ts`):** Відповідає за всі візуальні елементи (UI, барабани). Використовує контейнери PixiJS для побудови інтерфейсу.
- **Controller (`src/core/Controller.ts`):** Виступає сполучною ланкою. Асинхронно отримує дані через `SpinService`, оновлює Model і запускає відповідні візуальні зміни у View.
